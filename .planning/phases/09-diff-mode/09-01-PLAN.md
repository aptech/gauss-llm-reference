  <action>
    Create gauss-doc-qa/src/gauss_doc_qa/diff.py with three public functions:

    1. `parse_since(value: str) -> tuple[str, datetime | int]`
       - If value matches r/R + digits: return ("svn", int)
       - If value matches YYYY-MM-DD: parse with datetime.strptime, return ("date", datetime)
       - Otherwise: raise ValueError("--since must be a date (YYYY-MM-DD) or SVN revision (rNNNNN)")

    2. `filter_by_date(file_list: list[tuple[str, DocType]], cutoff: datetime) -> list[tuple[str, DocType]]`
       - For each (filepath, doc_type) in file_list, check os.path.getmtime(filepath)
       - Keep only files where mtime >= cutoff.timestamp()
       - Return filtered list (preserving sort order)

    3. `filter_by_svn_revision(file_list: list[tuple[str, DocType]], revision: int, docs_dir: str) -> list[tuple[str, DocType]]`
       - Run: subprocess.run(["svn", "diff", "--summarize", "-r", f"{revision}:HEAD", docs_dir], capture_output=True, text=True, check=True)
       - Parse output lines: each line has status char(s) then whitespace then absolute path
       - Build a set of changed absolute paths from svn output
       - Filter file_list to only entries whose filepath is in the changed set
       - On subprocess.CalledProcessError: raise RuntimeError with stderr message
       - On FileNotFoundError (svn not installed): raise RuntimeError("svn command not found")

    Write tests in test_diff.py:
    - Test parse_since with date, svn rev (upper/lower r), and invalid input
    - Test filter_by_date using tmp_path fixtures with controlled mtime (use os.utime)
    - Test filter_by_svn_revision with mocked subprocess.run (patch subprocess.run to return known output)
    - Test svn error handling (mock CalledProcessError, FileNotFoundError)
  </action>
