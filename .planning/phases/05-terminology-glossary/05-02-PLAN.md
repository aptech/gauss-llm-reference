   ```python
                 help="Path to YAML glossary file for terminology checking")
   ```

2. **Update scan function signature** to accept `glossary` parameter.

3. **Inside scan(), after the main checker loop, add glossary checking block:**
   ```python
   # Run glossary checker if --glossary provided
   if glossary:
       from gauss_doc_qa.glossary import load_glossary
       from gauss_doc_qa.checkers.glossary import GlossaryChecker

       glossary_entries = load_glossary(glossary)
       glossary_checker = GlossaryChecker(glossary_entries)

       for parsed in parsed_docs:
           findings = glossary_checker.check(parsed)
           all_findings.extend(findings)
   ```

4. **Important:** The glossary findings are appended to `all_findings` BEFORE `_render_findings` is called, so they automatically flow through terminal, JSON, and Markdown rendering (GLOS-03). No changes to report modules needed.
