  - Build alias_map via build_alias_map(glossary_entries)
  - For each finding with category == "terminology":

    - Read the file, get the original line at finding.line
    - Build fixed_text by replacing the non-canonical term with the canonical form. Use re.sub with word boundaries: `re.sub(r"\b" + re.escape(alias) + r"\b", canonical, original_line)`. Be case-aware: match the exact alias text found (not lowercased).
    - Confidence is always 1.0 (glossary aliases are exact matches, not fuzzy)
    - Create FixProposal with category="terminology", old_target=alias, new_target=canonical
