  4. **ai/schemas.py** -- Pydantic response models for structured Claude output:
     ```python
     from pydantic import BaseModel

     class CheckResult(BaseModel):
         check_id: str
         passed: bool
         evidence: str  # Quote from doc if failed, "" if passed
         line_hint: str  # Section/heading where issue found, "" if passed

     class PersonaReviewResponse(BaseModel):
         results: list[CheckResult]
     ```

  5. **ai/reviewer.py** -- Core review engine:
     ```python
     from gauss_doc_qa.models import Finding, ParsedDoc
     from gauss_doc_qa.ai.personas import Persona
     from gauss_doc_qa.ai.schemas import PersonaReviewResponse
     ```

     **`extract_doc_text(parsed_doc: ParsedDoc, persona: Persona) -> str`**:
     - Extract text content from ParsedDoc's raw_doc (docutils document).
     - Walk the docutils tree and extract text from paragraph, literal_block, and title nodes.
     - Strip RST directive syntax. Return plain text with section headings preserved.
     - For expert persona on COMMAND_REF pages: focus on extracting Format + Examples + Parameters sections if present.
     - For newcomer/writer: extract full page content (these pages are short).

     **`build_rubric_prompt(persona: Persona) -> str`**:
     - Format the rubric checks into a numbered list for the system prompt.
     - Each check formatted as: `{id}: {question}`
     - Include instruction: "For each check, determine pass or fail. If fail, quote the specific evidence from the document. If pass, leave evidence empty."

     **`run_persona_review(parsed_doc: ParsedDoc, persona: Persona) -> list[Finding]`**:
     - Extract doc text via `extract_doc_text()`.
     - Build system prompt from `persona.description` + `build_rubric_prompt()`.
     - Lazy import `anthropic` (inside function body, not at module level -- Pitfall 4 prevention).
     - Call `client.messages.parse()` with:
       - `model=persona.model`
       - `max_tokens=1024`
       - `temperature=0` (Pitfall 5 prevention)
       - `system=system_prompt`
       - `messages=[{"role": "user", "content": f"Document ({parsed_doc.path}):\n\n{doc_text}"}]`
       - `output_format=PersonaReviewResponse`
     - Convert response via `review_to_findings()`.

     **`review_to_findings(review: PersonaReviewResponse, persona: Persona, file_path: str) -> list[Finding]`**:
     - Build a map from check_id to RubricCheck.
     - For each result where `passed=False`:
       - Look up the RubricCheck by check_id.
       - Create a Finding with:
         - `file=file_path`
         - `line=None` (AI reviews cannot pinpoint exact lines)
         - `severity=check.fail_severity` (from rubric config, NOT from Claude)
         - `category=check.category`
         - `checker=f"ai_{persona.name}"` (e.g., "ai_newcomer", "ai_expert", "ai_writer")
         - `message=f"[{result.check_id}] {check.question} -- {result.evidence}"`
         - `auto_fixable=False`
     - Return list of Findings.
